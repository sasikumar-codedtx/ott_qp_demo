# Sony LIV Quickplay OTT Requirements HTML Package

Open `index.html` in a browser to review the complete one-document OTT product and API requirements guide.

Package contents:

- `index.html` - master requirement document with business flows, API details, rules, and implementation notes.
- `styles.css` - Stripe-inspired documentation styling.
- `app.js` - small progressive enhancement for active section highlighting and copy buttons.
- `assets/*.svg` - lightweight sample images and flow visuals used by the document.

The document is intentionally self-contained for product, design, API, and engineering review.
